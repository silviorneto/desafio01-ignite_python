# PyAgenda

Este aplicativo foi criado como parte do treinamento Ignite Python da Rocketseat (2024)

## A aplicação

O objetivo desse código é apresentar uma agenda para salvar, editar, deletar e marcar um contato como favorito. A interface utilizada é o próprio terminal.

## Requisitos

- python
- git

## Como utilizar o programa

01. Acesse o terminal e clone o projeto: `cd ~/app-agenda && git clone https://github.com/`
02. Execute o projeto: `python3 -m pygenda`

## Funcionalidades

- Cadastro de contato, incluindo nome, telefone, email e favorito.
- Lista contatos
- Edita contato
- Marca/desmarca contato como favorito
- Lista contatos favoritos
- Apaga um contato
